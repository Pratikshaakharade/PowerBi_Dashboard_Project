# 📊 Ecommerce Sales Dashboard

This project showcases an interactive Ecommerce Sales Dashboard used to analyze sales, profit, and customer insights.

## 🚀 Features

* Track Total Sales, Profit, Quantity, and AOV
* State-wise sales analysis
* Payment mode insights (COD, UPI, Cards)
* Monthly profit trends
* Category & sub-category performance

## 🛠️ Tools Used

* Power BI 

## 📂 Files Included

* Dashboard file
* Dataset
* Screenshot

## ▶️ How to Use

1. Download the file
2. Open in Power BI 
3. Explore filters and insights

## 📈 Insights

* Maharashtra has highest sales
* COD is most used payment method
* Clothing category dominates
